Verbs + idempotency

Idempotent = call N times, same effect as 1 time. Matters for retry logic — safe to retry idempotent requests; not safe for POST.

GET

Read resource. idempotent

POST

Create / non-idempotent action. NOT idempotent

PUT

Replace resource entirely. idempotent

DELETE

Remove resource. idempotent

PATCH

Partial update. usually idempotent
