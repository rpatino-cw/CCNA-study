ACL Builder + Test Harnessdrill

An ACL is a top-down checklist. Router walks lines until first match. Implicit deny at the end catches everything else. Standard = source IP only. Extended = source + dest + protocol + ports. Build, then fire test packets through.

Standard vs Extended

Standard ACL (1-99 / 1300-1999)

Source IP only. Place CLOSE TO destination — too generic to filter near source.

access-list 10 permit 10.1.1.0 0.0.0.255

Extended ACL (100-199 / 2000-2699)

Source + dest + protocol + ports. Place CLOSE TO source — drop bad traffic ASAP.

access-list 110 deny tcp 10.1.1.0 0.0.0.255 any eq 23

Wildcard mask vs Subnet mask

Subnet mask: 1 = network, 0 = host. Wildcard mask is the inverse: 0 = must-match, 1 = don't-care.

/24 subnet

Mask: 255.255.255.0
Wildcard: 0.0.0.255

/27 subnet

Mask: 255.255.255.224
Wildcard: 0.0.0.31
