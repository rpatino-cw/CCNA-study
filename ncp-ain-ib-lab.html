NCP-AIN · InfiniBand Fabric Diagnostics

IB Fabric Diagnostics Console

You're on a DGX H100 compute node (dgx-h100-01). The InfiniBand fabric is live. Use the diagnostic tools below to inspect HCA state, link health, fabric topology, and performance counters. Type help or ? for the command list.

admin@dgx-h100-01 — InfiniBand fabric
admin@dgx-h100-01:~$

Try these

  • ibstat HCA port state, physical state, rate, LID, GUIDs
  • ibstatus Terse per-port status summary
  • iblinkinfo Fabric-wide link map: speed, width, state per port
  • ibhosts List all CA (compute/host) nodes in the fabric
  • ibnetdiscover Full fabric topology: nodes, links, GUIDs
  • ibdiagnet Full fabric health scan: errors, BER, SM state
  • perfquery Port counters (Tx/Rx data+pkts, error counts)
  • sminfo Subnet manager info: GUID, priority, activity count
  • ibping <lid> Ping a fabric node by LID (e.g. ibping 4)
  • ibportstate <lid> <port> Physical port state for a specific LID+port
