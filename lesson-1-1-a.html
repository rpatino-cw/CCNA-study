What a Router Actually Doesdrill

A router is a Layer-3 device that connects different IP networks. It receives a frame, strips the L2 header, looks up the destination IP in its routing table, picks a next hop, rewrites the L2 header for that hop's interface, and ships it. Router vs. switch — what tells them apart

Router (Layer 3)

Connects different IP networks. Decides per-packet using destination IP. Each interface = its own subnet + its own broadcast domain. Runs routing protocols (OSPF). Lives in IP, ARP, ICMP, NAT, ACL.

Switch (Layer 2)

Connects devices in the same broadcast domain. Decides per-frame using destination MAC. All access ports = one VLAN/broadcast domain by default. Runs STP, CDP/LLDP, learns MACs. Has no IP smarts (unless it's a layer-3 switch).
