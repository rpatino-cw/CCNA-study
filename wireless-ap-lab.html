Wireless AP Lab — Placement

Drag access points onto the floor plan. Move, clone, delete — the RF heatmap recomputes live. Toggle layer / band in the sidebar.

Goal: cover every room with at least −67 dBm signal (voice grade). The colored heatmap is «max-RSSI of any AP» per square meter, attenuated by every wall the signal crosses. Concrete (−12 dB) blocks far more than drywall (−3 dB) or glass (−2 dB). Each AP’s 5 GHz radio drives the default heatmap — switch to 2.4 GHz to compare reach (longer range, more interference).

Drag a card to place. Drag a placed AP to move. Double-click (or right-click → Configure) opens the WLC-style modal — Radio / SSID·WLAN / Network / Advanced tabs. Switch the building preset top-left. Run a guided mission from the sidebar to learn step-by-step, or take a TEST mission to score yourself. Delete removes selected AP · Esc closes modal · localStorage persists everything.
