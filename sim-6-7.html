When to use which

JSON

REST APIs, JS apps, machine-to-machine. Strict syntax, minimal whitespace, can't comment. Most common in 2026.

YAML

Config files, Ansible, Kubernetes, CI pipelines. Whitespace-sensitive (tabs ≠ spaces). Comments OK. Human-readable.

XML

SOAP, NETCONF, legacy enterprise. Verbose, supports schemas (XSD) + namespaces. Heavyweight but rigorous.
