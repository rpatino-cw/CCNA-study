UFM Management Console

You are logged in to the UFM management node for a QM9790 InfiniBand fabric. UFM (Unified Fabric Manager) exposes a REST API at /ufmRest/ for topology, telemetry, and fabric events. Use GET and POST commands to query the API. Switch to Troubleshooting Scenarios to diagnose real fabric faults. Type help for the full endpoint list.

Try these

  • GET /ufmRest/resources/ports All fabric ports: GUID, LID, state, speed, width
  • GET /ufmRest/resources/links Fabric links: source/dest GUID, width, speed
  • GET /ufmRest/resources/systems Switches and HCAs: model, managed vs unmanaged
  • POST /ufmRest/monitoring/snapshot Create a one-time telemetry snapshot. Body: {"attributes":[...]}. Counters per port: errors, wait, discards
  • GET /ufmRest/app/events Fabric alarms and events (link down, threshold breach)
  • POST /ufmRest/actions Reset a port or trigger SM re-route
