Quick comparison

Forwarding plane / "what it does"

  • Router — L3, IP-based, between networks
  • L2 switch — L2, MAC-based, within a VLAN
  • L3 switch — both, fast inter-VLAN routing
  • NGFW / IPS — L3-7 inspection, stateful + signatures
  • AP — wireless ↔ wired bridge, MAC-level
  • Controller — orchestrates many APs / switches
  • Endpoint — originates / consumes traffic
  • Server — answers requests at L7
  • PoE — powers + carries data on one cable

Layer it lives at

  • Router → L3
  • L2 switch → L2
  • L3 switch → L2 + L3
  • NGFW → L3-L7
  • AP → L1-L2 (radio + bridge)
  • WLC / DNAC → management plane
  • Endpoint → L1-L7 stack
  • Server → L7 application
  • PoE → L1 power injection
