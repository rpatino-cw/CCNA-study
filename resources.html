Free Tools & Practice

Tool
Boson ExSim (Practice Exams)
The closest thing to the real CCNA exam. Paid ($99) but universally recommended. Has a free demo with limited questions.
$99 (free demo available) · Most realistic practice exam
Tool
cidr.xyz — Visual CIDR Calculator
Type any IP/CIDR and instantly see network, broadcast, range, and a bit-level breakdown showing which bits are network vs host. Updates as you type. 833 stars on GitHub.
Free · Instant visual feedback · Open source
Tool
Hilbert Curve IPv4 Map
Zoomable map of the entire IPv4 address space using a Hilbert curve. Zoom from the whole internet down to individual /24 blocks. Color-coded by IANA allocation. Inspired by xkcd #195.
Free · Interactive · D3.js · MIT license
Tool
Subnetting.org
Free online subnetting practice. Timed drills, random problems, instant feedback. Do 10 a day until it's automatic.
Free · Timed drills · Browser-based
Tool
CCNA Anki Flashcard Deck
Spaced repetition flashcards covering all CCNA objectives. Import into Anki and review daily. Pairs well with Jeremy's IT Lab deck.
Free · Anki (all platforms) · Spaced repetition
Tool
Synaptic Labs
Free browser-based network simulator. Alternative to Packet Tracer when you can't install software.
Free · Browser-based · No install
Tool
Visual Subnet Calculator
Enter a network, split and join subnets visually. Great for understanding VLSM by seeing the address blocks graphically.
Free · Visual VLSM tool
