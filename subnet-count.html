Core Formula

The Equation

One question type, one formula. "How many /26s fit in a /24?" reduces to subtraction and a power of two. Hover the colored parts to see what each number means.

subnets = 2(child − parent)
child > parent. Difference = bits borrowed = subnet count exponent.
child parent result

Why it works: every bit you borrow doubles the number of subnets. Borrow 1 bit → 2 subnets. Borrow 2 bits → 4. Borrow 3 → 8. The exponent is the count of bits you stole from host space and gave to network space.

Shortcut for the exam: just count the gap between the two prefix lengths and raise 2 to that power. /24 to /26 = gap of 2 = 22 = 4. Done.

Watch the Bits Borrow

The mask is just 32 bits. Start with /24 (24 network bits, 8 host bits). To go to /26, two host bits flip to network bits — those are the borrowed bits. Each borrowed bit doubles the subnet count. Click the buttons to step through.

= 4 subnets
