Subnet Showdown

Race a friend through subnet math. First to 4 wins.

Both players see the same IP and prefix. Type the answer the prompt asks for (network, broadcast, first/last usable, or usable host count). First correct gets the round. Best of 7 takes the match.

Pick a room

Share the room code with a friend. Same code = same room.

How answers are checked. For network/broadcast/first/last: type the dotted-quad — e.g. 172.16.40.0. For usable hosts: type the number — e.g. 1022. Whitespace and case don't matter. /31 and /32 only ask for network.
