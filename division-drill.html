Division Drill

When the magic number is small (2–32) and the IP is high (100–255), counting up the multiples is slow and error-prone. Drill the three fast moves: halve-down, chunk-subtract, and multiples ladder.

The pain. /28 means magic = 16. Last octet = 200. You don't want to count 16, 32, 48, 64, … all the way up. You want to land on 192 in one mental move. This drill builds that move.

Walkthrough
The three fast moves
  1. Halve-down. Magic is a power of 2. Divide IP by halving that many times. Magic 16 = halve 4 times. 200 → 100 → 50 → 25 → 12.5 → floor = 12. Block start = 12 × 16 = 192.
  2. Chunk-subtract. Subtract the biggest "fat" multiple of magic that fits, then handle the rest. Magic 16, IP 200: 16 × 10 = 160. 200 − 160 = 40. 16 × 2 = 32 fits. 160 + 32 = 192. Done.
  3. Multiples ladder. Anchor on the powers-of-2 multiples: ×1, ×2, ×4, ×8, ×16. Magic 16: 16, 32, 64, 128, 256. Land on the biggest ≤ IP, then add another magic if needed. 128 ≤ 200, 128 + 64 = 192 ≤ 200, 192 + 16 = 208 > 200 → block = 192.

Pick one and stick with it. Halve-down is fastest for /28–/30 (small magic). Chunk-subtract is better when you blank on multiples. Ladder is the safety net.
