What's different from CCNA?

Network+ is vendor-neutral — no Cisco IOS commands. It focuses on concepts, troubleshooting, and network design. The 13 CCNA-only topics you skip are: OSPF deep config (3.4), EtherChannel LACP (2.4), DTP/VTP (2.2 partial), WLC GUI (2.9), NAT config (4.1), SDN (6.2), SD architecture (6.3), AI/ML (6.4), REST APIs (6.5), Ansible/Terraform (6.6), and JSON (6.7).

Exam Format
Max 90 questions, 90 minutes. Multiple choice + PBQs. Passing score: 720/900.
PBQs on the Exam
Typically 5-7 PBQs. Drag-and-drop, matching, topology building, command selection. Skip them first, come back after multiple choice.
Key Domains
1. Networking Concepts (23%) 2. Implementation (19%) 3. Operations (16%) 4. Security (19%) 5. Troubleshooting (22%)

Exam Day Tips

1. Skip PBQs first. They appear at the beginning but take the most time. Flag them and do all multiple-choice first, then come back.
2. Time management: 90 questions in 90 minutes = 1 minute per question. Don't spend 5 minutes on one question.
3. Eliminate wrong answers. Remove obviously wrong options first. Even guessing between 2 options is 50/50.
4. Read the FULL question. CompTIA loves "MOST likely" and "BEST" and "FIRST step" — these qualifiers change the answer.
5. Troubleshooting methodology: If a question asks "what should you do next?", mentally identify which step you're on and pick the next step.
6. Physical layer first. When troubleshooting, always check cables/connections before software. CompTIA rewards bottom-up OSI thinking.
7. Passing score: 720/900. You don't need 100%. You can miss ~20% and still pass. Don't panic over hard questions.
