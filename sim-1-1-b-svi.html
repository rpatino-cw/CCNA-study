Trap to remember

  • up/up ≠ routing. ip routing is OFF by default on Catalyst.
  • SVI needs VLAN in db + a live member port (autostate) to come up.
  • ip default-gateway = L2-only mgmt, not an SVI. Hosts use the SVI IP as gateway.
