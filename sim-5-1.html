Security Foundations Simdrill

Every defender ranks Confidentiality, Integrity, Availability by context — wrong rank = wrong control. Then trace the kill-chain: threat → vulnerability → exploit → mitigation. Same four words, every breach.

Why CIA priority changes

A bank cares integrity first — if balances flip, money vanishes even if site stays up. Streaming cares availability — buffering kills brand. Health records care confidentiality — leak is felony. Pick a scenario, rank C/I/A from most to least critical, then check.

Threat → Vuln → Exploit → Mitigation

Threat = the actor or thing that could harm. Vulnerability = the weakness it could ride in on. Exploit = the actual technique used. Mitigation = the control you bolt on. SQL injection example below — click each step.
