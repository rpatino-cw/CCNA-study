IPv6 Anycast Routing Sim

Four servers all share 2001:db8::dead:beef. Send a packet — routing picks the nearest one. Drag the client around to see routing change in real time.
WHY this exists. Anycast lets multiple servers advertise the same address. Routing protocols pick whichever instance has the lowest cost path. Same address = automatic load distribution + failover. DNS root servers, Cloudflare 1.1.1.1, Google 8.8.8.8 — all anycast. From the client's view, there's one IP. From the network's view, the closest server wins.
