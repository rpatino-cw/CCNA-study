Config Bootcamp

Two phases per topic. Guided teaches the exact CLI in order. Recall makes you type it from memory, like your life depends on it.

Pick a topic. Phase A walks the config line by line with the why, the core truth, and the traps. Phase B hands you a scenario and a blank terminal. Type the config from memory and the checker grades it line by line, then makes you nail the verify command. Highest yield topics come first.
