Topology architectures (1.2)2-tier collapses core+distribution into one layer; 3-tier keeps access, distribution, core separate.
Cabling types (1.3)Straight-through = unlike devices (PC-switch); crossover = like devices (switch-switch).
IP params per OS (1.10)Windows: ipconfig. Mac/Linux: ifconfig (or ip addr). DHCP gives IP, mask, gateway, DNS.
Virtualization / VMs + containers (1.12)VM = full OS on a hypervisor; container shares the host OS kernel (lighter, faster).
TFTP / FTP (4.9)TFTP = UDP 69, no auth (IOS/config transfer); FTP = TCP 20/21, has auth.
Security program elements (5.2)Three elements: user awareness, training, physical access control.
Password policy / MFA / certs / biometrics (5.4)MFA = two of: something you know, have, are. Biometric = are.
IPsec / VPN (5.5)AH = integrity only; ESP = encryption + integrity. IKE builds the tunnel.
SDN / controller architecture (6.1-6.3)Northbound API = controller-to-apps (REST); Southbound API = controller-to-devices (OpenFlow).
REST APIs (6.5)CRUD verbs: GET=read, POST=create, PUT=update, DELETE=remove. Stateless over HTTP.
Ansible / Terraform / config mgmt (6.6)Ansible = agentless, push, SSH/YAML; Puppet/Chef = agent, pull.
JSON components (6.7)Key:value pairs. {} = object, [] = array. Strings need quotes; commas separate.
AI / ML in network ops (6.4)AI = mimic human reasoning; ML = learns from data (subset of AI); both drive predictive/automated ops.
