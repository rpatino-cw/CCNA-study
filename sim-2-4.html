Spanning Tree Sim

Three switches in a triangle = a Layer 2 loop. Without STP, broadcasts multiply forever and the network melts. STP elects a root, declares which port wins on each segment, and blocks the rest. Watch the storm, then enable STP and watch the tree emerge.

Why STP exists

Ethernet has no TTL. A broadcast frame in a loop circles forever and gets duplicated at every junction — bandwidth saturates in seconds, MAC tables flap, switches lock up. STP (802.1D) prevents this by computing a spanning tree across all switches: one root, one path from every switch to the root, all redundant links blocked. RSTP (802.1w) is the modern default — same idea, sub-second convergence. Root election: lowest Bridge ID wins, where BID = (priority, MAC). Default priority is 32768 — the tiebreaker is usually MAC, which is why the oldest switch often wins by accident.
