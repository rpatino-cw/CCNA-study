One Box, Many Worldsdrill

Virtualization splits one physical resource into many isolated logical ones — reducing cost, simplifying provisioning, and increasing density. VMs virtualize the whole machine. Containers virtualize the user space. VRFs virtualize a router's routing table.

Bare metal = one OS owns the hardware. Hypervisor = each VM has its OWN kernel (heavy). Containers = SHARE the host kernel (light, fast).

VRF — Virtualized routing table

VRFs let one router host overlapping IP ranges (10.0.0.0/8 in two tenants) without leaking traffic between them. Each interface lives in exactly one VRF.
