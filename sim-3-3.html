Static Routing Sim

Type real Cisco syntax. Watch the routing table update. Add a default. Add a floating backup. Then kill the primary link and watch the floating take over.
WHY this matters Static routes are the operator's promise to the router: "If you see this destination, send it here." No protocol, no neighbors, no convergence — just a hardcoded entry. The interesting variants: default route catches all unknowns; host route (/32) targets one IP; floating static sits dormant with a high AD until the primary disappears, then activates. Static is the only way to guarantee a path that doesn't move under your feet.
