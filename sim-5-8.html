WPA2 4-way handshake vs WPA3 SAE

WPA2 — 4-way handshake

  1. AP → ANonce
  2. Client → SNonce + MIC
  3. AP → GTK
  4. Client → ACK
Capture handshake → offline dictionary attack on PSK.

WPA3 — SAE (Dragonfly)

  1. Both sides → Commit (PW Element)
  2. Both sides → Confirm
  3. Mutually authenticated
  4. Forward secrecy: each session = new keys
Offline brute-force = NOT possible. Each guess needs live AP.
