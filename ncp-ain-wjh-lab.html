NCP-AIN · WJH Drop Analysis and NetQ

WJH Drop Console

You are on the NetQ management server monitoring a Spectrum-X leaf-spine fabric. WJH (What Just Happened) captures hardware-accelerated drop telemetry from Mellanox/NVIDIA Spectrum ASICs. Use netq show wjh-drop to inspect L1/L2/Router/Buffer/ACL/Tunnel drop events. Use netq check roce to validate the 5 RoCE sub-tests. Type help for the full command list.

netq@netq-mgmt -- WJH drop analysis console
netq@netq-mgmt:~$

Try these

  • netq show wjh-drop All hardware drop events across L1/L2/Router/Buffer/ACL/Tunnel
  • netq show wjh-drop l1 Physical-layer drops: signal loss, FEC, link down
  • netq show wjh-drop l2 L2 drops: STP, MAC errors, VLAN mismatch
  • netq show wjh-drop router L3 drops: TTL, MTU, no route, ACL deny
  • netq show wjh-drop buffer Buffer drops: tail drop, PFC watchdog, headroom overflow
  • netq show wjh-drop acl ACL-hit drops: ingress/egress rule matches
  • netq show wjh-drop tunnel Tunnel drops: decap errors, VXLAN miss, ECMP
  • netq check roce 5-subtest RoCE validation: mode, classification, congestion, flow-control, ETS
  • netq show roce-counters Per-switch RoCE hardware counters
