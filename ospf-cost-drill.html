How OSPF cost works (recipe)
  1. Per-link cost: cost = ref_bw / interface_bw (integer divide, floor). Minimum 1.
  2. Default ref_bw on Cisco: 100 Mbps. So FastEth, GigE, 10GE all return cost 1 — every modern link looks the same. Fix: auto-cost reference-bandwidth 100000 (set to 100 Gbps).
  3. Path cost = sum of outgoing interface costs along the path. Don't double-count ingress.
  4. Best path = lowest total cost. Ties go to multiple equal-cost routes (default 4-way ECMP).
  5. Bandwidth-from-target: if you want a specific cost N at default ref_bw, interface_bw = 100/N (in Mbps). Cost 4 = 25 Mbps; cost 64 = 1.5625 Mbps (T1 ≈ 1.544 Mbps → cost 64).
