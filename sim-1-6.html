IPv4 Subnetting Calculator

Type an IP and CIDR. The 32 bits split into network (amber) and host (gray). Network address, broadcast, usable hosts, dotted-decimal mask — all updated live.
WHY this exists. A subnet mask divides 32 bits into two parts. Network bits identify which network the IP belongs to. Host bits identify which device on that network. Routers route on network bits. ANDing the IP with the mask gives the network address. Subtracting from broadcast gives usable hosts. Once you can see the bits, the math becomes obvious.
