Forwarding Decision Sim

Step through what a router actually does when a packet arrives. Six stages: dest lookup → next-hop → recursive lookup → ARP → encap → TTL. Toggle "next-hop unreachable" to see ICMP fire.
WHY this matters Routing-table lookup picks where to send the packet — but the router still has to wrap it in a Layer-2 frame to actually transmit. The destination MAC has to come from somewhere: ARP. The next-hop IP from the route table is rarely on a directly-connected segment, so the router does a recursive lookup to find the egress interface. Get this six-step pipeline and you'll never wonder again why ARP timeouts cause packet loss.
