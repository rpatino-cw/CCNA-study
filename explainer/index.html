The 5-layer architecture

Every interaction flows through exactly 5 layers. Hover any layer to see its job.

1. USER
Browser navigates to a page. Click, type, scroll. No login.
2. HTML PAGE
83 standalone pages. Each loads its own scripts via <script defer>. Shared nav injected by nav.js. Examples: core.htmlstrat.htmlquiz.htmlsubnetting-mastery.html
3. JS LAYER
Vanilla IIFEs attaching globals to window. No bundler, no modules. Examples: storequiz-enginemastery-trackerfsrsstrat
4. JSON DATA
Read-only datasets fetched once per page. The big ones are questions.json (415K), term-mcqs.json (1.6MB), topics.json (40K). Loaded as data/*.js globals or fetch() calls.
5. localStorage
Every key prefixed ccna_. Holds proficiency (EMA), mastery state machine, quiz history, streak, study time, settings. Survives reload. Export/import via Settings.
