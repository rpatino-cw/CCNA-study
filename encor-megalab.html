Platform recommendation

BEST CHOICE
Cisco CML 2 Personal ($199/yr) — bundles licensed Cat9000v IOS-XE so eBGP + route-maps, VRF, VXLAN/LISP, and the automation domain (NETCONF/RESTCONF/Python) all work day one with no image-sourcing friction. ~16 GB RAM minimum, 32 GB recommended for the full 14-node topology.
Runner-up

EVE-NG Community (free) — works, but you must source IOS-XE images legally via Cisco license or DevNet. More setup friction.

Supplement only

Boson NetSim — confirmed gaps: BGP basics only (no route-maps), no NETCONF/RESTCONF, no GRE/VXLAN. Use for command drills, not deep labs.

Supplement only

Jeremy's IT Lab ENCOR (YouTube) — free and solid, but work-in-progress. Not a complete spine yet.

Do not use

Packet Tracer — no BGP route-maps, no VXLAN/LISP, no automation. Not suitable for ENCOR depth.

12-week cadence

Week Parts Focus
W1P1 + P2L2 foundation: MST, EtherChannel, HSRP/VRRP
W2P3OSPF multi-area: LSA types, ABR, NSSA, auth
W3P4 + P5 startEIGRP named mode + beginning redistribution
W4P5 + P6 startRedistribution loop prevention + eBGP bring-up
W5P6BGP path manipulation: LOCAL_PREF, MED, 13-step, NAT
W6P7Overlays: GRE, IPsec VTI, VXLAN, LISP concept
W7P8QoS: LLQ, NBAR, NTP sync
W8P9Security: 802.1X, MAB, CoPP, uRPF, ACL
W9P10Assurance: NetFlow, IP SLA, SNMPv3, SPAN/RSPAN
W10P11Automation: NETCONF, RESTCONF, Ansible, EEM
W11AllFull-lab troubleshooting: break and fix each domain
W12AllReview weak spots + Boson ExSim mock exams
~10-12 hours/week. Hardest material (redistribution + BGP) is front-loaded in weeks 3-5. Note: ENCOR v1.2 removed wireless objectives, so no AP node is in this topology.
