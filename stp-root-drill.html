Walkthrough
How root election works (recipe)
  1. Bridge ID = 2-byte priority + 6-byte MAC (8 bytes total). Modern Cisco: priority field is 4-bit prio + 12-bit ext system ID (VLAN). So priority on the wire = configured priority + VLAN.
  2. Default priority = 32768. For VLAN 10 → BID priority field = 32768 + 10 = 32778.
  3. Lowest bridge ID wins. Compare priority first; if tied, compare MAC (lowest MAC wins).
  4. Priority must be a multiple of 4096 when set manually (because of the 12-bit ext system ID). Valid: 0, 4096, 8192, 12288, 16384, 20480, 24576, 28672, 32768, ...
  5. Root path cost = sum of incoming-link costs from this bridge back to root. IEEE 802.1D-2004 long costs: 10G=2000, 1G=20000, 100M=200000, 10M=2000000.
