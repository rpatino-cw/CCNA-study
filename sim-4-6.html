DHCP Relay Sim — ip helper-address

Routers don't forward broadcasts by default. The DHCP Discover is broadcast — so a server one subnet away never sees it. Solution: configure `ip helper-address` on the SVI, and the router rewrites the broadcast as a unicast aimed at the server. The server sees giaddr (gateway IP) and knows which scope to offer from.
