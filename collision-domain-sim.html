Collision Domain Simulator

Watch frames travel down a shared bus, hear the JAM signal when two stations talk over each other, and see how a switch makes the problem disappear by giving every port its own collision domain.

The one-line answer

A hub = one collision domain shared by every port. If two devices transmit at once, the signals overlap, the hub re-amplifies the mess, every device hears garbage, JAM signal goes out, everyone backs off.

A switch = every port is its own collision domain. Frames are buffered, MAC tables learned, ports run full-duplex — collisions become physically impossible. That's why hubs died.

Hub mode L1 repeater

Hub copies every bit it sees on every port — it's a multi-port repeater. No memory, no MAC table, no buffering.

  • Two stations transmit → signals overlap on the wire → collision.
  • Stations detect overlap (voltage spike), send a JAM signal so everyone knows.
  • Each station picks a random backoff (0..2^k - 1 slot times, capped at k=10) and retries.
  • Throughput collapses fast as load grows. ~30-40% utilization is the practical ceiling.

Switch mode L2 bridge

Switch reads the destination MAC, looks it up in its CAM table, and forwards the frame only out the matching port.

  • Each port = its own collision domain → collisions impossible in full-duplex.
  • Multiple conversations run in parallel (PC-A→PC-B and PC-C→PC-D simultaneously, no contention).
  • Broadcast frames (FF:FF:FF:FF:FF:FF) still flood every port — broadcast domain unchanged.
  • Routers (L3) are what break broadcast domains, not switches.

The exam shortcut

  • Count collision domains = count switch ports + every hub-side bus once.
  • Count broadcast domains = count router interfaces / VLANs.
  • Hub adds 0 collision domains (it is one). Switch adds N. Router adds N broadcast domains.
  • Mixed mode shows it: 3 switch-direct PCs + 1 hub bus = 4 collision domains, still 1 broadcast domain.

CSMA/CD in 4 steps

  • CS — Carrier Sense: listen before transmit.
  • MA — Multiple Access: anyone may try.
  • CD — Collision Detect: keep listening while sending.
  • On collision → JAM (32 bits of garbage) → binary exponential backoff → retry. Modern switched + full-duplex links disable CSMA/CD entirely.
