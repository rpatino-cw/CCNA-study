BGP

BGP 13-Step Best Path Selection

Heavy hitters are highlighted. In Flash Mode, click a step to reveal the hint.

1 Highest Weight Cisco-proprietary, local to the router only. Not advertised. Default 0.
2 Highest Local Pref Shared within the AS via iBGP. Default 100. Higher = preferred.
3 Locally originated Prefer routes originated on this router (network/aggregate/redistribute) over learned routes.
4 Shortest AS-Path Fewer AS hops = preferred. Most commonly manipulated attribute.
5 Lowest Origin code IGP (i) < EGP (e) < Incomplete (?). IGP is best (network statement).
6 Lowest MED Multi-Exit Discriminator. Suggests entry point to neighbors in adjacent AS. Compared only between same-AS peers by default.
7 eBGP over iBGP External BGP routes preferred over internal BGP routes.
8 Lowest IGP metric to next-hop Prefer the path whose next-hop is closest in the IGP table.
9 Oldest eBGP route Stability: prefer the route received longest ago (eBGP only).
10 Lowest Router-ID BGP Router-ID of the advertising peer.
11 Lowest cluster-list length Route Reflector environments only.
12 Lowest neighbor IP Tiebreaker: prefer session with numerically lowest peer IP.
EIGRP

EIGRP Key Facts

Parameter Value Notes
K1 (Bandwidth) 1 Enabled by default. Contributes to classic metric.
K2 (Load) 0 Disabled by default.
K3 (Delay) 1 Enabled by default. Cumulative delay along path.
K4 (Reliability) 0 Disabled by default.
K5 0 Disabled by default.
Classic metric formula (10^7 / min-BW + sum-delay) x 256 BW in kbps; delay in tens of microseconds
Feasible Distance (FD) Best metric Lowest calculated metric to destination, stored in topology table
Feasible Successor (FS) RD < FD Feasibility Condition: neighbor's Reported Distance must be less than this router's FD

QoS

QoS / DSCP Values

Class / Name DSCP Value Typical Use / Notes
EF (Expedited Forwarding) 46 Voice bearer (RTP). Requires LLQ. Strict priority queue.
CS6 48 Network control traffic (routing protocols, management)
AF41 34 Interactive video (videoconferencing)
AF31 26 Signaling (call control)
AF11 10 Bulk data (backup, FTP)
Default / BE 0 Best-effort. No QoS treatment.
QoS Mechanism Behavior Notes
AFxy notation x=class, y=drop x = class 1-4 (higher = better); y = drop precedence 1 (low) to 3 (high)
Policing Drop / Remark No buffer. Excess traffic dropped or re-marked immediately.
Shaping Buffer + Delay Excess traffic held in queue and sent later. Introduces delay.

STP / MST

STP / MST Facts

Parameter Value Notes
Bridge priority default 32768 Lower value = more likely to become root bridge
Priority increment 4096 Must be a multiple of 4096 (0 to 61440)
RSTP port roles Root / Designated / Alternate / Backup Alternate = replacement root port; Backup = redundant designated port
RSTP port states Discarding / Learning / Forwarding Collapsed from STP's 5 states. Discarding = Blocking + Listening.
MST default revision 0 Region revision number; must match on all switches in the region
MST default instance Instance 0 IST (Internal Spanning Tree). All VLANs map here until remapped.
