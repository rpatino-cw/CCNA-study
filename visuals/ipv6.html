SLAAC vs DHCPv6

SLAAC (Stateless Address Autoconfiguration)

  • Router sends RA (Router Advertisement) with prefix
  • Host generates Interface ID via EUI-64 or random
  • No DHCP server needed
  • Host gets: prefix + interface ID = full GUA
  • Uses ICMPv6 RS/RA messages

DHCPv6 Stateful

  • Server assigns the full IPv6 address
  • Like DHCP for IPv4
  • Server tracks which addresses are assigned
  • RA flag: M=1 (Managed)
  • Can also provide DNS, domain name, etc.

DHCPv6 Stateless

  • SLAAC provides the IPv6 address
  • DHCPv6 provides DNS, domain name, etc.
  • Server does NOT track addresses
  • RA flag: O=1 (Other config)
  • Best of both: simple address + extra config

Neighbor Discovery Protocol (NDP)

NDP replaces ARP, ICMP Router Discovery, and ICMP Redirect in IPv6. It uses ICMPv6 messages.

DAD Process: Before using any IPv6 address, the host sends an NS to its own tentative address (solicited-node multicast). If no NA is received back, the address is unique and safe to use. If an NA comes back, there is a duplicate and the address is not assigned.

IPv6 Configuration Commands

Global & Interface Configuration
ipv6 unicast-routing ! Enable IPv6 routing globally
 
interface GigabitEthernet0/0
ipv6 address 2001:db8:1::1/64 ! Assign GUA
ipv6 address fe80::1 link-local ! Assign link-local
ipv6 address autoconfig ! Use SLAAC
ipv6 enable ! Enable IPv6, auto-generates link-local
no shutdown
Show Commands
show ipv6 interface brief ! Quick view of IPv6 addresses on all interfaces
show ipv6 interface GigabitEthernet0/0 ! Detailed IPv6 info for an interface
show ipv6 route ! IPv6 routing table
show ipv6 neighbors ! NDP neighbor cache (like ARP table)
show ipv6 protocols ! IPv6 routing protocols
Static Routing
ipv6 route ::/0 2001:db8:1::1 ! Default route via next-hop
ipv6 route ::/0 GigabitEthernet0/1 ! Default route via exit interface
ipv6 route 2001:db8:2::/64 2001:db8:1::1 ! Static route to subnet

IPv6 Address Shortening Rules

Rule 1
Remove Leading Zeros

Leading zeros in each 16-bit group can be omitted. Each group must keep at least one digit.

2001:0db8:0042:0000:0000:0000:0000:0001
↓
2001:db8:42:0:0:0:0:1
Rule 2
Replace Consecutive All-Zero Groups with ::

One consecutive group of all-zero groups can be replaced with :: but only once per address!

2001:db8:42:0:0:0:0:1
↓
2001:db8:42::1
