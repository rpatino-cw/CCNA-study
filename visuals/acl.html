Top-Down Processing

  • ACLs are evaluated top to bottom, one ACE at a time
  • First match wins - once matched, no further ACEs are checked
  • Order of entries matters enormously

Implicit Deny

Every ACL has an invisible deny ip any any at the end. If no ACE matches, the packet is dropped.

Standard ACLs (1-99, 1300-1999)

  • Match source IP only
  • Place close to destination
  • Less specific = keep away from source to avoid blocking valid traffic

Extended ACLs (100-199, 2000-2699)

  • Match source IP, destination IP, protocol, and port
  • Place close to source
  • More specific = safe to filter early

Inbound vs Outbound

Inbound: ACL checked before routing decision.
Outbound: ACL checked after routing decision, before packet exits.

Standard ACL Numbered: 1-99, 1300-1999

! Standard ACL — matches source IP only access-list 10 permit 192.168.1.0 0.0.0.255 access-list 10 deny 10.0.0.0 0.255.255.255 ! Apply to interface (close to DESTINATION) interface Gi0/0 ip access-group 10 out

Extended ACL Numbered: 100-199, 2000-2699

! Extended ACL — matches src IP, dst IP, protocol, port access-list 100 permit tcp 192.168.1.0 0.0.0.255 host 10.1.1.1 eq 80 access-list 100 permit icmp any any access-list 100 deny ip any any log ! Apply to interface (close to SOURCE) interface Gi0/0 ip access-group 100 in

Best Practices

Standard ACLs: Place close to the DESTINATION — they only match source IP, so placing them near the source would block traffic to all destinations.
Extended ACLs: Place close to the SOURCE — they are specific enough (protocol, port, destination) to filter accurately without collateral damage.
Order matters: Put the most specific entries first. A broad deny any before specific permits will block everything.
Implicit deny: Always remember there is a hidden deny ip any any at the end. If you only have deny statements, nothing gets through.
Use remark: Document your ACLs with remarks — future-you will thank present-you.
Use named ACLs: They are easier to edit (you can delete individual entries) and far more readable than numbered ACLs.
