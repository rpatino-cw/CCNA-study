Modified EUI-64 Calculator

Generate a 64-bit IPv6 interface ID from a 48-bit MAC. Insert fffe in the middle. Flip the universal/local bit (bit 7 of byte 1). Done.
WHY this exists. SLAAC (stateless address autoconfiguration) lets a host self-generate an IPv6 address. The router advertises the /64 prefix; the host needs a 64-bit interface ID. EUI-64 derives that ID from the MAC: split MAC in half, insert fffe, flip bit 7 (universal/local) so a globally-unique MAC becomes locally-unique IID. Modern OSes mostly use random IIDs for privacy, but EUI-64 still appears on routers, IoT, and exam questions.
