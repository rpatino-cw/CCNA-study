An interactive, client-side study guide that maps directly to the official Cisco CCNA 200-301 exam blueprint. Track your progress across all 97 objectives, take quizzes, run diagnostics, and build real skills with Packet Tracer labs — all from your browser with zero setup.

![CCNA Study System](screenshot.png)

## Features

- **97 exam objectives** with per-objective quizzes and proficiency tracking
- **6 domain coverage** matching the official Cisco blueprint weighting
- **Diagnostic assessment** to map strengths and weaknesses before you start
- **Cheat sheets** and quick-reference material for every topic
- **Difficulty stats** and spaced-repetition-aware review suggestions
- **Packet Tracer labs** — hands-on configuration exercises (VLANs, OSPF, NAT, ACLs, and more)
- **Interactive visualizers** for subnetting, STP, OSPF, ACLs, IPv6, and VLANs
- **Progress tracking** with export/import — all stored in localStorage
- **Exam date countdown** and study strategy planner
- **Knowledge map** showing your coverage across all domains at a glance

## Getting Started

**[Open the study system](https://rpatino-cw.github.io/CCNA-study/)** and click **Take Diagnostic** to begin. The diagnostic maps your current knowledge, then the dashboard builds a personalized study plan targeting your weakest areas first.

## Built With

- HTML, CSS, vanilla JavaScript
- No frameworks, no backend, no build step
- All progress stored client-side in localStorage
- Deployed via GitHub Pages
