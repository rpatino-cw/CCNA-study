Game Night

One room, all the games. Host picks what everyone plays.

Start a room, share the link, and switch between any of the 9 multiplayer drills on the fly. Scores stack across every game in the session — first to score 1 in a game still feels like a win, but the running total is the real bragging right.

How to host. Click Start Room, share the link. Once friends join, click any game button — everyone's screen jumps to that game. Switch mid-round any time. Reset Scores wipes the leaderboard for a fresh tournament. If you joined first by accident and the wrong person became host, anyone can Claim host after the host has been idle 30 seconds.
